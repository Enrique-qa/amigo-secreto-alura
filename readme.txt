Um projeto simples para sortear nomes de amigos secretos.

Projeto para o Challenge de lógica e programação para o progama Oracle ONE

## Como funciona

Digite o nome de um amigo no campo de texto.
Clique em "Adicionar" para incluir o nome na lista.
Repita o processo para adicionar mais nomes.
Clique em "Sortear amigo" para escolher um nome aleatório da lista.

## Tecnologias usadas

HTML
CSS
JavaScript

## Como executar

Baixe ou clone este repositório.
Abra o arquivo index.html no seu navegador.
Adicione nomes antes de sortear
